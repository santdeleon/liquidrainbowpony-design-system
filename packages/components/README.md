# @liquidrainbowpony/components

> Custom React components built with the Styled Components library

## Usage

The Liquidrainbowpony Design System uses React and Styled components to create beautiful, fully functioning, eclectic components out of the box. All you need to is install and import them and they'll work as expected.

## Getting Started

1. run `npm install` or `yarn install` to install the necessary dependencies.
2. import any components needed with the `import { ButtonPrimary } from '@liquidrainbowpony/components'` syntax.
3. Done!
