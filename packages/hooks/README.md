# @liquidrainbowpony/hooks

> The React hook library for the Liquidrainbowpony Design System

## Usage

The Liquidrainbowpony Design System uses custom React hooks to help theme components, handle theme toggling, and assist with component functionality. As well, you can import any of the custom hooks standalone from this package to use in any use-case that fits.

## Getting Started

1. run `npm install` or `yarn install` to install the necessary dependencies.
2. import the react hooks as needed.
3. Done!
